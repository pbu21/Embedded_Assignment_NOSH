### Demonstration

The video demonstrates:
- Initial LED OFF state
- 1st button press → LED blinks at 0.5 Hz
- 2nd button press → LED blinks at 1 Hz
- 3rd button press → LED blinks at 2 Hz
- 4th button press → LED turns OFF
- 5th button press → Cycle repeats
- Low-power mode operation
